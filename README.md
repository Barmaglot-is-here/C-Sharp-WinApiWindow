# C# WinApi Window
Простое окно, созданное при помощи WinApi и C#. 

Обладает следующими функциями: 
- Maximize
- Minimize
- Restore
- Hide
- Show
- Close
- Fullscreen

Имеет изменяемые: 
- Размер
- Позицию
- Заголовок

Эвенты:
- OnSizeChanged
- OnPositionChanged
- OnKeyDown
- OnKeyUp
